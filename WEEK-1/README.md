# Week 1 Assignments - JavaScript Basics

This folder contains JavaScript programs that helped me practice the basic concepts of programming and problem solving using JavaScript.

## Assignments List

### Day 1: Basic Logic and Arrays

#### GreaterNumber.js
Finds the bigger number between two given numbers using if-else conditions.

#### GreaterNumber_in_given3.js
Finds the biggest number among three numbers using conditional statements.

#### SumofArray.js
Calculates the sum of all elements present in an array using loops.

#### 4SmallestElementInArray.js
Finds the smallest element in an array by comparing elements.

#### func_returns_bigNumber.js
A function that receives three numbers as arguments and prints the biggest number.

#### func_returnSum.js
A function that receives an array as argument and returns the sum of all elements.

#### Search_inarray.js
Searches for a given element in an array and prints its index position.  
If the element is not present, it prints `"not found"`.

Operations performed:
- Insert a new employee at the 2nd position
- Remove employee with name `"Kiran"`
- Update Sneha's marks from `95` to `75`
